# MineChat 本地开发环境配置与运行指南

> 更新日期：2026-06-29

---

## 一、前置条件

| 软件 | 版本要求 | 说明 |
|------|---------|------|
| Node.js | ≥ 18.x | LTS版本 |
| pnpm | ≥ 9.x | 包管理器 |
| PostgreSQL | ≥ 15 | 数据库 |
| Redis | ≥ 7 | 缓存（可选，初期可不用） |

---

## 二、克隆项目

```bash
git clone git@github.com:hyuzz-nuc/MineChat.git
cd MineChat
```

---

## 三、后端配置

### 3.1 安装依赖

```bash
cd server
pnpm install
```

> ⚠️ pnpm 11.x 首次安装需运行 `pnpm approve-builds` 交互式授权原生模块

### 3.2 配置环境变量

```bash
cp .env.example .env
```

编辑 `.env` 文件，确认以下配置：

```env
PORT=3000
DATABASE_URL=postgresql://postgres:你的密码@localhost:5432/minechat
JWT_SECRET=你的密钥
JWT_REFRESH_SECRET=你的刷新密钥
CLIENT_URL=http://localhost:5173
```

### 3.3 创建数据库

```bash
# PostgreSQL 中创建数据库
createdb minechat
```

### 3.4 初始化数据库

```bash
# 生成 Prisma Client
pnpm db:generate

# 推送 Schema 到数据库（开发阶段）
pnpm db:push

# 填充种子数据
pnpm db:seed
```

### 3.5 启动后端

```bash
pnpm dev
```

启动成功后会看到：
```
🚀 MineChat server running on http://localhost:3000
📡 Socket.IO ready on the same port
```

## 四、前端配置

### 4.1 安装依赖

```bash
cd client
pnpm install
```

### 4.2 环境配置

`.env` 文件已预置开发环境配置：

```env
VITE_API_BASE_URL=http://localhost:3000/api/v1
VITE_SOCKET_URL=http://localhost:3000
```

### 4.3 启动前端

```bash
pnpm dev
```

启动成功后访问 http://localhost:5173

## 六、项目结构

```
MineChat/
├── client/                 # 前端 (Vue3 + Vite)
│   ├── src/
│   │   ├── api/           # API请求模块
│   │   ├── composables/   # 组合式函数
│   │   ├── router/        # 路由配置
│   │   ├── stores/        # Pinia状态管理
│   │   ├── styles/        # 设计系统CSS
│   │   └── views/         # 页面组件
│   └── .env               # 前端环境变量
├── server/                 # 后端 (Express + TypeScript)
│   ├── src/
│   │   ├── config/        # 配置
│   │   ├── controllers/   # 控制器
│   │   ├── middleware/     # 中间件
│   │   ├── routes/        # 路由
│   │   ├── services/      # 业务逻辑
│   │   ├── socket/        # WebSocket处理
│   │   ├── validators/    # 请求验证
│   │   └── utils/         # 工具函数
│   ├── prisma/            # 数据库Schema & 种子
│   └── .env               # 后端环境变量
└── docs/                   # 项目文档
```

## 七、常用命令

### 后端

```bash
cd server
pnpm dev          # 开发模式启动
pnpm build        # 编译TypeScript
pnpm db:generate  # 生成Prisma Client
pnpm db:push      # 推送Schema到数据库
pnpm db:migrate   # 创建迁移
pnpm db:seed      # 填充种子数据
pnpm db:studio    # 打开Prisma Studio
```

### 前端

```bash
cd client
pnpm dev          # 开发模式启动
pnpm build        # 构建生产版本
pnpm preview      # 预览生产构建
```
