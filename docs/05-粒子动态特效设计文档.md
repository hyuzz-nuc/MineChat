# 05 - 粒子动态特效设计文档

> 版本: v1.0 | 日期: 2026-06-29 | 状态: 设计中

## 1. 设计目标

参照 MineRadio 的 Three.js 粒子系统，为 MineChat 聊天界面打造**轻量级 Canvas 2D 粒子动态背景**，实现沉浸式暗色空间感，同时保持聊天功能的性能与可读性。

### 1.1 与 MineRadio 的区别

| 维度 | MineRadio | MineChat |
|------|-----------|----------|
| 渲染引擎 | Three.js WebGL | Canvas 2D (轻量) |
| 粒子用途 | 专辑封面3D展示 + 歌词粒子 + 手势交互 | 聊天背景氛围增强 |
| 粒子数量 | ~14000+ (封面) | 200-400 (背景) |
| 交互 | 拖拽旋转 + 手势推动 + 鼠标跟随 | 鼠标跟随 + 自动漂浮 |
| 性能要求 | GPU密集 | 低CPU占用，不影响聊天 |

## 2. 粒子系统架构

### 2.1 粒子类型

```
┌──────────────────────────────────────────────┐
│  MineChat 粒子系统                            │
├──────────┬──────────┬───────────┬────────────┤
│ 深空星尘 │ 光点漂浮 │ 连线网络   │ 呼吸脉冲   │
│ 120-200  │ 30-60    │ 动态生成   │ 5-8        │
│ 远景小点 │ 中景光球 │ 近距连线   │ 节奏呼吸   │
│ 缓慢飘移 │ 随机轨迹 │ 透明度渐变 │ accent色   │
└──────────┴──────────┴───────────┴────────────┘
```

### 2.2 粒子属性

```typescript
interface Particle {
  // 位置
  x: number;          // 0~1 归一化坐标
  y: number;          // 0~1 归一化坐标
  z: number;          // 深度 0~1 (0=远 1=近)
  
  // 运动
  vx: number;         // x方向速度
  vy: number;         // y方向速度
  speed: number;      // 基础速度系数
  
  // 外观
  size: number;       // 基础半径 0.5~3px
  opacity: number;    // 基础透明度 0.1~0.6
  hue: number;        // 色相 (accent色域内浮动)
  
  // 生命周期
  phase: number;      // 呼吸相位偏移
  seed: number;       // 随机种子
}
```

### 2.3 渲染层次

```
z-index: 0  → Canvas粒子背景 (position:fixed, inset:0)
z-index: 1  → SVG噪点纹理叠加
z-index: 10 → 毛玻璃UI层 (sidebar + chat area)
z-index: 50 → 模态框/弹窗
```

## 3. 视觉效果规范

### 3.1 色彩

| 元素 | 颜色 | 说明 |
|------|------|------|
| 深空星尘 | `rgba(0,245,212,0.08~0.25)` | accent色微光 |
| 光点漂浮 | `rgba(255,255,255,0.15~0.35)` | 暖白光点 |
| 连线 | `rgba(0,245,212,0.03~0.08)` | 极淡accent连线 |
| 呼吸脉冲 | `rgba(0,245,212,0.02~0.06)` | 大范围柔和光晕 |

### 3.2 动画参数

| 参数 | 值 | 说明 |
|------|---|------|
| 帧率 | 30fps (节能) | requestAnimationFrame + 节流 |
| 粒子速度 | 0.002~0.01/frame | 非常缓慢的漂浮 |
| 呼吸周期 | 4~8秒 | sin曲线透明度变化 |
| 连线距离 | <120px | 两粒子距离小于阈值才连线 |
| 鼠标影响半径 | 200px | 鼠标附近粒子轻微偏移 |
| 鼠标影响力度 | 0.3px/frame | 非常轻微的推力 |

### 3.3 性能预算

| 指标 | 目标 |
|------|------|
| CPU占用 | <5% (单核) |
| 内存 | <5MB |
| 粒子总数 | 200-400 |
| 每帧绘制 | <2ms |
| 帧率 | 稳定30fps |

## 5. 交互设计

### 5.1 鼠标跟随

- 鼠标移动时，附近粒子轻微被"推开"
- 推力随距离衰减 (反比平方)
- 鼠标离开后粒子缓慢回归原轨迹
- 效果极其微妙，不影响UI操作

### 5.2 状态响应

| 应用状态 | 粒子行为 |
|----------|----------|
| 正常聊天 | 标准漂浮 |
| 收到新消息 | 短暂脉冲加速 (0.5s) |
| 对方正在输入 | accent色粒子亮度微增 |
| 空闲(>5min无操作) | 粒子减速至50% |

## 6. 降级策略

| 条件 | 降级 |
|------|------|
| 移动端 | 粒子数量减半，无连线 |
| 低性能设备 | 仅保留深空星尘，无光点 |
| prefers-reduced-motion | 完全禁用粒子，静态背景 |
| 后台标签页 | 暂停动画循环 |
