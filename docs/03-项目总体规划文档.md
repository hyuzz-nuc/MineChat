## 一、项目愿景

**MineChat** — 一款追求极致视觉与交互品质的 Web 端实时聊天软件。

采用沉浸式暗色设计语言，将毛玻璃、微光效、丝滑动效融入聊天场景，打造"深空对话"般的高级感通讯体验。

## 三、API接口规范预览

### 3.1 RESTful API 设计原则

- URL使用小写+连字符：`/api/v1/friend-requests`
- 使用HTTP状态码：200/201/400/401/403/404/500
- 统一响应格式：

```json
{
  "code": 0,
  "message": "success",
  "data": { }
}
```

### 3.2 核心API清单

| 方法 | 路径 | 描述 |
|------|------|------|
| POST | /api/v1/auth/register | 用户注册 |
| POST | /api/v1/auth/login | 用户登录 |
| POST | /api/v1/auth/refresh | 刷新Token |
| GET | /api/v1/users/me | 获取当前用户 |
| PUT | /api/v1/users/me | 更新个人资料 |
| POST | /api/v1/users/me/avatar | 上传头像 |
| GET | /api/v1/users/search?q= | 搜索用户 |
| GET | /api/v1/friends | 好友列表 |
| POST | /api/v1/friend-requests | 发送好友申请 |
| PUT | /api/v1/friend-requests/:id | 处理好友申请 |
| GET | /api/v1/conversations | 会话列表 |
| GET | /api/v1/messages/:roomId | 历史消息 |
| POST | /api/v1/groups | 创建群组 |
| PUT | /api/v1/groups/:id | 更新群信息 |
| POST | /api/v1/groups/:id/members | 添加群成员 |
| DELETE | /api/v1/groups/:id/members/:userId | 移除群成员 |

### 3.3 WebSocket 事件清单

| 事件 | 方向 | 描述 |
|------|------|------|
| `message:send` | Client→Server | 发送消息 |
| `message:new` | Server→Client | 收到新消息 |
| `message:read` | Client→Server | 标记已读 |
| `message:read-ack` | Server→Client | 已读回执 |
| `typing:start` | Client→Server | 开始输入 |
| `typing:stop` | Client→Server | 停止输入 |
| `typing:indicator` | Server→Client | 输入指示 |
| `presence:online` | Server→Client | 用户上线 |
| `presence:offline` | Server→Client | 用户离线 |
| `friend:request` | Server→Client | 好友申请通知 |
| `friend:accepted` | Server→Client | 好友申请通过 |
