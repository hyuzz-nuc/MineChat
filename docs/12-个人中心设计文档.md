# 12-个人中心设计文档

> 参照 Discord / Telegram / QQ 等主流聊天软件的个人中心设计方案

## 1. 设计理念

MineChat 个人中心遵循「沉浸式毛玻璃 + 深空主题」的视觉体系，采用侧边弹出面板（类似Discord）而非独立页面（类似Telegram），原因：

- **不打断聊天流**：Discord式侧面板让用户可以边聊天边查看/编辑个人资料
- **视觉一致性**：与现有好友面板共享同一侧边栏区域，交互逻辑统一
- **高级感**：毛玻璃背景 + 粒子穿透效果，延续MineRadio的沉浸感

## 2. 主流方案对比

| 软件 | 交互方式 | 信息层级 | 编辑方式 |
|------|----------|----------|----------|
| **Discord** | 左侧弹出面板 | 头像+用户名+标签(#UID)+状态+签名 | 内联编辑 |
| **Telegram** | 侧滑全屏页面 | 头像+昵称+UID+手机号+签名 | 独立编辑页 |
| **QQ** | 弹窗卡片 | 头像+昵称+QQ号+签名+等级 | 弹窗内编辑 |
| **WeChat** | 独立页面 | 头像+昵称+微信号+地区 | 独立页编辑 |

**MineChat选型**：Discord式侧边弹出面板 + 内联编辑

## 3. 功能清单

### 3.1 信息展示
- **头像**（圆形，支持hover放大）
- **昵称**（大号文字）
- **UID**（#100001 格式，accent色，点击复制）
- **用户名**（@username 格式，灰色副标题）
- **个性签名**（bio字段，灰色小字）
- **在线状态**（在线/离线/忙碌，带呼吸灯）
- **注册时间**（格式化日期）

### 3.2 信息编辑
- **昵称编辑**（点击昵称旁编辑按钮，内联输入框）
- **个性签名编辑**（点击签名旁编辑按钮，内联textarea）
- **头像更换**（点击头像，弹出文件选择器，后续可扩展）
- **在线状态切换**（下拉选择：在线/离线/忙碌）

### 3.3 操作入口
- **复制UID**（点击UID数字，toast提示"已复制"）
- **退出登录**（底部按钮，红色强调）

## 5. 交互细节

- **打开**：点击导航栏头像 → 侧面板从左滑入（300ms ease-out-expo）
- **关闭**：点击←按钮 / 点击面板外区域 / Esc键
- **UID复制**：点击UID文字 → `navigator.clipboard.writeText()` → toast "UID已复制"
- **内联编辑**：点击✏️ → 文字变为input → Enter保存 / Esc取消
- **状态切换**：点击状态区域 → 下拉菜单选择 → Socket广播 `presence:update`

## 7. 样式规范

- 面板宽度：280px（与好友面板一致）
- 背景：glass-panel毛玻璃效果
- 动画：slideInLeft 300ms ease-out-expo
- UID字体：monospace, accent色(#00F5D4), 11px
- 编辑按钮：✏️ emoji, opacity 0→1 on hover
