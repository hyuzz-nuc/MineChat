# 14-开机动画与登录页设计文档 V2

## 1. V1问题与V2改进方向

### V1问题
1. **文字重叠**："Mine"和"Chat"都用 `position:absolute; top:50%; left:50%; transform:translate(-50%,-50%)` 导致完全重叠
2. **缺乏动感**：两个词同时出现，没有先后节奏感
3. **颜色不够高级**：纯accent绿(#00F5D4)缺少质感和层次

### V2改进方案

| 问题 | V2方案 |
|------|--------|
| 文字重叠 | "Mine"和"Chat"并排排列，用inline-block自然流布局 |
| 缺乏动感 | "Mine"从左侧滑入 → 0.5s后 → "Chat"从右侧滑入 → 合并碰撞 → 弹性回弹 → 鎏金波纹扫过 |
| 颜色不够高级 | 鎏金渐变色系：底色暗金#C9A84C → 亮金#F4D28A → 白金#FFF8E1 → 玫瑰金#E8B4B8，配合accent绿#00F5D4作为对比色 |

## 2. V2时间轴（总计 5秒，更紧凑更有节奏）

```
0.0s ─ 0.6s    : 黑屏→深色背景渐入，尘埃粒子开始飘动
0.6s ─ 1.8s    : 光束线条扫描，碎片从中心扩散
1.2s ─ 2.0s    : 信号线展开+波形线
2.0s ─ 2.6s    : "Mine"从左侧滑入（带skewX倾斜+弹性overshoot）
2.5s ─ 3.1s    : "Chat"从右侧滑入（带skewX反向倾斜+弹性overshoot）
3.1s ─ 3.3s    : 两词碰撞合拢，微弹跳
3.3s ─ 3.8s    : 鎏金波纹从左到右扫过文字（shimmer效果）
3.8s ─ 4.0s    : 闪光脉冲（全屏暖金闪光）
4.0s ─ 4.5s    : 信号线脉冲，副标题淡入
4.5s ─ 5.0s    : "点击进入"提示出现（呼吸脉冲）
5.0s+           : 用户交互 → 退出过渡 → 登录界面
```

## 3. 品牌文字动画详细设计

### 3.1 布局

```
┌──────────────────────────────┐
│                              │
│        Mine  Chat            │ ← 并排，中间间距8px
│       ──────────             │ ← 信号线
│     real-time immersive chat │ ← 副标题
│        点击进入               │
└──────────────────────────────┘
```

"Mine"和"Chat"用 `display:inline-block`，不用 `position:absolute`，自然并排。

### 3.2 "Mine"入场动画

```css
@keyframes splash-mine-in {
  0% {
    opacity: 0;
    transform: translateX(-120px) skewX(-12deg) scaleX(1.15);
    letter-spacing: .08em;
  }
  40% {
    opacity: 1;
    transform: translateX(12px) skewX(2deg) scaleX(.97);
    letter-spacing: -.02em;
  }
  60% {
    transform: translateX(-6px) skewX(-1deg) scaleX(1.02);
  }
  80% {
    transform: translateX(2px) skewX(0) scaleX(.99);
  }
  100% {
    opacity: 1;
    transform: translateX(0) skewX(0) scaleX(1);
    letter-spacing: -.04em;
  }
}
```

**效果**：从左侧高速飞入 → 略微超过目标位置 → 弹性回弹2次 → 稳定

### 3.3 "Chat"入场动画

```css
@keyframes splash-chat-in {
  0%, 25% {
    opacity: 0;
    transform: translateX(120px) skewX(12deg) scaleX(1.15);
    letter-spacing: .08em;
  }
  55% {
    opacity: 1;
    transform: translateX(-12px) skewX(-2deg) scaleX(.97);
    letter-spacing: -.02em;
  }
  75% {
    transform: translateX(6px) skewX(1deg) scaleX(1.02);
  }
  90% {
    transform: translateX(-2px) skewX(0) scaleX(.99);
  }
  100% {
    opacity: 1;
    transform: translateX(0) skewX(0) scaleX(1);
    letter-spacing: -.04em;
  }
}
```

**效果**：比"Mine"晚0.5s，从右侧飞入 → 弹性回弹 → 与"Mine"并排

### 3.4 鎏金波纹（Shimmer）

文字出现后，一道鎏金光泽从左到右扫过：

```css
.splash-wordmark {
  position: relative;
  overflow: hidden;
}
.splash-wordmark::after {
  content: '';
  position: absolute;
  top: 0; left: -100%; width: 60%; height: 100%;
  background: linear-gradient(
    105deg,
    transparent 0%,
    rgba(244,210,138,0) 30%,
    rgba(255,248,225,0.7) 48%,
    rgba(244,210,138,0.5) 52%,
    rgba(244,210,138,0) 70%,
    transparent 100%
  );
  animation: splash-shimmer 1.2s cubic-bezier(.22,1,.36,1) 3.3s forwards;
}

@keyframes splash-shimmer {
  0% { left: -100%; opacity: 0 }
  20% { opacity: 1 }
  100% { left: 200%; opacity: 0 }
}
```

## 4. 鎏金色彩体系

### 4.1 主色板

| 色彩 | Hex | 用途 |
|------|-----|------|
| 暗金 | `#C9A84C` | "Mine"文字底色，沉稳质感 |
| 亮金 | `#F4D28A` | "Chat"文字渐变主色，鎏金光泽 |
| 白金 | `#FFF8E1` | 波纹高光，闪光脉冲 |
| 玫瑰金 | `#E8B4B8` | 微妙的暖色点缀 |
| accent绿 | `#00F5D4` | 信号线、波形线，冷色对比 |
| 深空黑 | `#010304` | 背景基色 |
| 碳灰 | `#08090B` | 过渡色 |

### 4.2 "Mine"文字样式

```css
.splash-word-mine {
  color: #C9A84C;
  text-shadow:
    0 0 22px rgba(201,168,76,.35),     /* 暗金辉光 */
    0 22px 72px rgba(0,0,0,.58),        /* 深度阴影 */
    -2px 0 0 rgba(0,245,212,.22),       /* 左侧accent绿边缘光 */
    2px 0 0 rgba(244,210,138,.18);      /* 右侧亮金边缘光 */
}
```

### 4.3 "Chat"文字样式

```css
.splash-word-chat {
  background: linear-gradient(
    100deg,
    rgba(255,255,255,.04) 0%,
    #FFF8E1 22%,
    #F4D28A 42%,
    #C9A84C 58%,
    #00F5D4 78%,
    rgba(255,255,255,.72) 100%
  );
  background-size: 300% 100%;
  -webkit-background-clip: text;
  background-clip: text;
  color: transparent;
  -webkit-text-fill-color: transparent;
  animation: splash-chat-gradient 3s ease-in-out infinite alternate;
}

@keyframes splash-chat-gradient {
  0% { background-position: 0% 50% }
  100% { background-position: 100% 50% }
}
```

**效果**：文字本身是流动的鎏金→绿→白金渐变，缓慢流动。

## 5. 引擎增强

### 5.1 粒子颜色更新

dust粒子新增暖金色粒子（20%概率），不再纯白：
```
rgba(244,210,138, alpha)  — 20%
rgba(201,168,76, alpha)   — 10%
rgba(255,255,255, alpha)  — 70%
```

### 5.2 光束颜色更新

streak光束改为鎏金主导：
```
rgba(244,210,138,...)  — 40%  亮金
rgba(201,168,76,...)   — 30%  暗金
rgba(0,245,212,...)    — 20%  accent绿
rgba(232,180,184,...)  — 10%  玫瑰金
```

### 5.3 闪光颜色更新

闪光脉冲改为暖金色：
```
0%: rgba(201,168,76,0)
35%: rgba(244,210,138,0.28*flash)
50%: rgba(255,248,225,0.42*flash)
65%: rgba(244,210,138,0.28*flash)
100%: rgba(201,168,76,0)
```
