## 一、设计理念

### 1.1 核心设计哲学

MineChat 的视觉语言基于沉浸式暗色设计体系，并针对聊天场景进行适配。核心理念可概括为：

> **「深空对话」** — 在无垠暗色空间中，每一则消息都如同星光般清晰而温暖。

三大设计支柱：

| 支柱 | 含义 | 视觉映射 |
|------|------|----------|
| **深邃** | 暗色基底承载内容 | 纯黑背景 + 深色层级 |
| **通透** | 毛玻璃传递层次 | backdrop-filter 模糊 + 半透明 |
| **微光** | 细节处点缀光感 | accent色发光 + 边框微光 |

### 1.2 设计风格特征

| 维度 | 特征描述 | MineChat 适配 |
|------|-----------|---------------|
| 主背景 | 纯黑 #000 + 3D粒子 | 纯黑 #08090B + 微光渐变 |
| 主强调色 | 青绿 #00F5D4 | 青绿 #00F5D4（继承） |
| 辅助色 | 香槟金 #f4d28a | 冰蓝 #73a7ff（更科技感） |
| 毛玻璃 | 控制栏/面板 | 侧栏/消息气泡/弹窗 |
| 字体 | Noto Sans SC | Noto Sans SC（继承） |
| 圆角 | 大圆角 50px/22px | 中大圆角 24px/16px |
| 动效 | 丝滑 cubic-bezier | 继承，聊天场景略收敛 |

### 2.2 毛玻璃体系

基于沉浸式暗色设计的毛玻璃体系，根据聊天场景定义三个层级：

```css
/* 层级1：侧栏 - 最强模糊 */
--glass-sidebar:
  background: rgba(14,16,20,.72);
  backdrop-filter: blur(40px) saturate(1.4) brightness(1.1);
  border-right: 1px solid rgba(0,245,212,.12);
  box-shadow: inset -1px 0 0 rgba(255,255,255,.04),
              4px 0 24px rgba(0,0,0,.20);

/* 层级2：面板 - 中等模糊 */
--glass-panel:
  background: linear-gradient(112deg, rgba(72,74,76,.62), rgba(24,27,30,.70) 48%, rgba(8,12,14,.74));
  backdrop-filter: blur(34px) saturate(1.34);
  border: 1px solid rgba(0,245,212,.30);
  box-shadow: 0 22px 64px rgba(0,0,0,.30),
              0 0 34px rgba(0,245,212,.052),
              inset 0 1px 0 rgba(255,255,255,.16);

/* 层级3：弹窗/浮层 - 轻模糊 */
--glass-overlay:
  background: rgba(12,12,18,.78);
  backdrop-filter: blur(24px) saturate(1.12);
  border: 1px solid rgba(255,255,255,.09);
  box-shadow: 0 18px 54px rgba(0,0,0,.38),
              inset 0 1px 0 rgba(255,255,255,.07);
```

## 五、交互设计

### 5.1 页面切换动效

```
会话切换：
  旧会话 → opacity: 1→0, translateX(0→-12px), 200ms
  新会话 → opacity: 0→1, translateX(12px→0), 280ms ease-out-expo

面板展开/收起：
  展开 → opacity: 0→1, translateY(12px→0), scale(.98→1), 280ms
  收起 → opacity: 1→0, translateY(0→-8px), scale(1→.98), 200ms
```

### 5.2 消息发送动效

```
新消息出现：
  1. 消息气泡从下方滑入 + 淡入
     translateY(16px→0), opacity(0→1), 280ms ease-out-expo
  2. 气泡出现后微微回弹
     scale(1→1.01→1), 120ms

自己发送消息：
  1. 消息立即显示（乐观更新）
  2. 发送中：消息右侧显示旋转加载点
  3. 发送成功：加载点变为单勾 ✓
  4. 送达：单勾变双勾 ✓✓（灰色）
  5. 已读：双勾变为accent色 ✓✓
```

### 5.3 微交互清单

| 场景 | 动效 | 参数 |
|------|------|------|
| 按钮hover | 微上浮 + 背景亮起 | translateY(-1px), 180ms |
| 会话项hover | 背景渐现 + 微浮 | translateY(-1px), 200ms |
| 输入框focus | 边框accent发光 + 微浮 | translateY(-1px), 280ms |
| 未读徽章 | 脉冲发光 | 2s infinite alternate |
| 在线状态变化 | 指示灯呼吸 | 1.5s ease-in-out |
