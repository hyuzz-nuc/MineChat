# 20 - 演播厅界面UI设计文档

> 编写日期：2026-07-01 | 版本：v1.0

---

## 一、设计理念

演播厅界面延续MineChat V3金属金视觉体系，以**专业演播室**为设计隐喻：

- **深色调底色**：`#08090B`主背景 + 毛玻璃层级叠加，营造影院级沉浸感
- **金属金点睛**：`#D4AF37`作为强调色贯穿控制条、弹幕文字、按钮高亮
- **毛玻璃分层**：侧栏(Sidebar) → 面板(Panel) → 弹窗(Overlay)三级毛玻璃
- **流畅过渡**：所有交互均有`--ease-out-expo`缓动曲线，200-450ms过渡

## 二、页面布局

```
┌─────────────────────────────────────────────────┐
│  ← 返回    演播厅名称(金属渐变)    👥3    ⚙️    │  ← 顶部栏 glass-panel

│    + DanmakuCanvas       │  │                │  │
│                          │  ├────────────────┤  │
│    ┌──────────────────┐  │  │ 输入框  [发送]  │  │
│    │  弹幕输入 [发送]  │  │  └────────────────┘  │
│    └──────────────────┘  │                      │
│    ▶ ━━━●━━━━━ 12:30     │                      │
```

### 响应式断点
- **≥768px**：左右分栏布局（视频70% + 侧边栏30%）
- **<768px**：上下堆叠（视频 + 固定240px聊天区），隐藏拖拽分割线

## 三、核心组件

### 3.1 VideoPlayer 视频播放器

| 元素 | 实现 |
|------|------|
| 视频元素 | HTML5 `<video>`，`object-fit: contain`，16:9比例 |
| 弹幕层 | Canvas 2D叠加，`pointer-events: none` |
| 控制条 | 3秒无操作自动隐藏，hover渐显 |
| 进度条 | 4px高度，hover放大至6px + 金色滑块 |
| 弹幕输入 | 顶部圆角输入框 + 金属渐变发送按钮 |
| 缓冲指示 | 金色旋转spinner |

### 3.2 DanmakuEngine 弹幕引擎

| 特性 | 参数 |
|------|------|
| 渲染 | Canvas 2D @ 60fps |
| 轨道数 | 8条滚动轨道 + 3条顶部固定 + 3条底部固定 |
| 弹幕颜色 | `#D4AF37`（金属金），系统消息`#8A9099` |
| 描边 | `rgba(0,0,0,.6)` 2px黑色描边增强可读性 |
| 最大同屏 | 80条 |
| 速度 | 2.5px/frame ± 20%随机 |
| 无障碍 | `prefers-reduced-motion`自动禁用 |

### 3.3 ChatSidebar 聊天侧边栏

| 功能 | 实现 |
|------|------|
| Tab切换 | 聊天/成员双标签，金色下划线指示 |
| 消息列表 | 同ChatView左右分侧布局，系统消息居中 |
| 输入区 | 圆角输入框 + 金属渐变发送按钮 |
| 成员列表 | 头像 + 昵称 + 角色标签(房主/管理) + 在线状态灯 |

### 3.4 TheaterView 主视图

| 功能 | 实现 |
|------|------|
| 顶部栏 | glass-panel毛玻璃，返回/房间名(金属渐变文字)/成员数/设置 |
| 视频源弹窗 | Teleport至body，毛玻璃遮罩+缩放淡入动画 |
| 侧边栏拖拽 | `mousedown/move/up`调整宽度，240-480px范围 |
| Socket同步 | theater:play/pause/seek/sync/chat/danmaku事件 |

## 四、动效规范

| 交互 | 动效 | 时长 | 缓动 |
|------|------|------|------|
| 控制条显隐 | opacity渐变 | 280ms | ease-out-expo |
| 弹幕飘入 | fadeInUp | 250ms | ease-out-expo |
| 弹窗出现 | scaleIn | 300ms | ease-out-expo |
| 弹窗遮罩 | fadeIn | 200ms | ease-out-expo |
| 进度条hover | height放大 | 180ms | ease-out-expo |
| 按钮hover | translateY(-1px) | 180ms | ease-out-expo |
| Tab切换 | border-bottom-color | 180ms | ease-out-expo |
